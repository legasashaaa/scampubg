ВАЖНАЯ ИНФОРМАЦИЯ ДЛЯ ПОЛУЧЕНИЯ ПРИЗА

Как получить приз?

Для получения выигранного приза необходимо:

  • 1. Войти через социальную сеть - выберите один из предложенных способов авторизации
  • 2. Указать ID своего аккаунта - введите ваш игровой ID для зачисления приза (обязательный шаг!)
  • 3. Подтвердить получение - нажмите кнопку "Получить приз"

После этих действий приз будет отправлен на вашу игровую почту в течение 24 часов.
